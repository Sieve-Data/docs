---
title: "Get Model"
api: "GET /v1/models/{model_id}"
---

This endpoint returns a JSON object that describes a given model given its ID.

### Request
<ParamField path="model_id" type="string" required>
  name of Sieve model
</ParamField>

### Response
<ResponseField name="id" type="string">
  The id of the model.
</ResponseField>
<ResponseField name="name" type="string">
  The name of the model.
</ResponseField>
<ResponseField name="status" type="string">
  The status of the model. The possible values are "ready", "queued", "building", and "failed".
</ResponseField>
<ResponseField name="inputs" type="dict">
  The inputs of the model as a dictionary of name and type.
</ResponseField>
<ResponseField name="outputs" type="dict">
  The outputs of the model as a dictionary of name and type.
</ResponseField>

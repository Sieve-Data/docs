---
title: "Get Model Requests"
api: "GET /models/{model_id}/requests"
---

This endpoint returns a JSON object that describes the requests sent to a model directly.

### Request
<ParamField path="model_id" type="string" required>
  Sieve model ID
</ParamField>

<ParamField query="page" type="number" default="1">
  The page of results to return
</ParamField>

<ParamField query="limit" type="number" default="100">
  The number of results to return per page
</ParamField>

### Response
<ResponseField name="id" type="string">
  The id of the job.
</ResponseField>
<ResponseField name="model_id" type="string">
  The id of the model that the job belongs to.
</ResponseField>
<ResponseField name="time" type="dict">
  The times when the job was queued, started, and completed.
</ResponseField>
<ResponseField name="status" type="string">
  The status of the job. Can be "queued", "started", "finished", or "error".
</ResponseField>
<ResponseField name="user_inputs" type="dict">
  The inputs that the user provided to trigger the job.
</ResponseField>

---
title: "sieve deploy"
---

Deploy functions and models to Sieve.

## Usage
```bash
sieve deploy [OPTIONS] PATH
```

## Options
- `--help` Show this message and exit.

## Arguments
- `PATH` Path to files containing functions and models to deploy.

## Example
Deploy a function defined in `test.py`:
```bash
sieve deploy test.py
```

Deploy every function and model defined in a directory:
```bash
sieve deploy
```

---
title: "Get One Function Usage"
api: "GET /v2/usage/function/{organization_name}/{function_name}"
---

### Request
This endpoint retrieves usage statistics for your's organization usage of a specific function, filtered by the provided start and end timestamps.

<ParamField path="organization_name" type="string" required>
  name of Sieve organization.
</ParamField>
<ParamField path="function_name" type="string" required>
  name of Sieve model.
</ParamField>
<Snippet file="usage_parameters.mdx" />

### Response

<ResponseExample>
```json
{
  "cpu": 3.298063308383466,
  "n1-standard-4-T4": 0.18991428048098052,
  "total_cost": 3.4879775888644464
}
```
</ResponseExample>

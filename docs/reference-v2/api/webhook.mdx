### Webhook Response

The webhook responses vary depending on the event.

### Event `job_complete`

<ResponseField name="type" type="string">
The type of event. Always `job_complete`.
</ResponseField>
<ResponseField name="body" type="object">
The event data.
<ResponseField name="status" type="string">
The status of the job.
</ResponseField>
<ResponseField name="job_id" type="string">
The job ID.
</ResponseField>
<ResponseField name="output" type="object">
(Deprecated) The first output of the job. Output field will be null if the job errored out.
<Expandable title="properties">
<Snippet file="output.mdx" />
</Expandable>
</ResponseField>
<ResponseField name="outputs" type="array[object]">
A list of all outputs emitted by the job, if any.
<Expandable title="properties">
<Snippet file="output.mdx" />
</Expandable>
</ResponseField>
<ResponseField name="error" type="string">
Will be null if no error occurred.
</ResponseField>
</ResponseField>

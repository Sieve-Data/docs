---
title: "List Function Versions"
api: "GET /v2/functions/{organization_name}/{function_name}/versions"
---

This endpoint returns a list of function versions of a function. 

### Request
<ParamField path="organization_name" type="string" required>
  name of Sieve organization
</ParamField>
<ParamField path="function_name" type="string" required>
  name of Sieve model
</ParamField>
<ParamField query="limit" type="string" default="none">
  Limit the number of response, used for pagination
</ParamField>
<ParamField query="offset" type="string" default="none">
  The offset of the response, used for pagination
</ParamField>

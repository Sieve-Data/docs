---
title: "Get Function Version"
api: "GET /v2/functions/{organization_name}/{function_name}/versions/{id}"
---

This endpoint returns a JSON object that describes a given function version given its ID.

### Request
<ParamField path="organization_name" type="string" required>
  name of Sieve organization
</ParamField>
<ParamField path="function_name" type="string" required>
  name of Sieve model
</ParamField>
<ParamField path="id" type="string" required>
  version id of Sieve function 
</ParamField>

---
title: "Get Function Version"
api: "GET /v2/functions/{organization_name}/{function_name}/versions/{id}"
---

This endpoint returns a JSON object that describes a given function version given its ID.

### Request
<ParamField path="organization_name" type="string" required>
  name of Sieve organization
</ParamField>
<ParamField path="function_name" type="string" required>
  name of Sieve model
</ParamField>
<ParamField path="id" type="string" required>
  version id of Sieve function 
</ParamField>

### Response
<ResponseField name="id" type="string">
  Id of the model.
</ResponseField>
<ResponseField name="build_status" type="string">
  Build status of the function version. Statuses are 'queued', 'building', 'pushing', 'deploying', 'ready', 'error'
</ResponseField>
<ResponseField name="queued_at" type="string">
  The time that the model build is queued.
</ResponseField>
<ResponseField name="built_at" type="string">
  The time that the model build finished building.
</ResponseField>
<ResponseField name="ready_at" type="string">
  The time that the model is ready to process jobs.
</ResponseField>
<ResponseField name="compute_type" type="string">
  Compute Type (TODO: Finalize the types)
</ResponseField>
<ResponseField name="python_packages" type="string array">
  An array of the model's python pip packages.
</ResponseField>
<ResponseField name="system_packages" type="string array">
  An array of the model's system packages.
</ResponseField>
<ResponseField name="cuda_version" type="string">
  Cuda Version.
</ResponseField>
<ResponseField name="gpu" type="boolean">
  Whether this model uses gpu.
</ResponseField>
<ResponseField name="minimum_replicas" type="integer">
    The minimum number of replicas of the model. Scale your model's minimum replicas to avoid cold starts.
</ResponseField>
<ResponseField name="maximum_replicas" type="integer">
    The maximum number of replicas of the model. Prevents model replicas from scaling beyond a certain point.
</ResponseField>

---
title: "Get Job"
api: "GET /v2/jobs/{id}"
---

This endpoint returns a JSON object with information about the job along with its outputs.

### Request
<ParamField path="id" type="string" required>
  id of the job
</ParamField>

### Response
<ResponseField name="id" type="string">
  The id of the job.
</ResponseField>
<ResponseField name="function_id" type="string">
  The id of the function that the job belongs to.
</ResponseField>
<ResponseField name="function" type="string">
  The name of the function that the job belongs to.
</ResponseField>
<ResponseField name="status" type="string">
The status of the job. Possible values: 'queued', 'processing', 'error', 'finished', 'cancelled';
</ResponseField>
<ResponseField name="created_at" type="string">
  The time the job was created.
</ResponseField>
<ResponseField name="started_at" type="string">
  The time the job started processing by the function.
</ResponseField>
<ResponseField name="completed_at" type="string">
  The time the job was completed.
</ResponseField>
<ResponseField name="inputs" type="list">
  TODO: Copy paste in input type after its finalized
</ResponseField>
<ResponseField name="outputs" type="list">
  TODO: Copy paste in output type after its finalized
</ResponseField>
<ResponseField name="error" type="string">
  The error that the job ran into. Empty string if there's no error. 
</ResponseField>
<ResponseField name="visibility" type="string">
  "public" or "private". The error that the job ran into. Empty string if there's no error. 
</ResponseField>

---
title: "Get Project Info"
api: "GET https://api.sievedata.com/v1/get_project_info"
---

Get all info about a given project including `fps` which defines the frame rate your project runs at, `batch_size` which defines to processing parallelism, `tagThis returns the tag schema, the FPS, the pipeline configuration, and other fields of the project.

### Parameters

<Param query="project_name" type="string" required>
  The name of the project
</Param>

<RequestExample>

```bash Terminal
curl https://api.sievedata.com/v1/get_project_info \
  -H "X-API-Key: YOUR_API_KEY"
```

</RequestExample>

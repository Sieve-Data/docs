---
title: "Delete Project"
api: "POST https://api.sievedata.com/v1/delete_project"
---

### Parameters

<Param body="project_name" type="string" required>
  The name of the project
</Param>

<Param body="delete_data" type="string" required>
  Whether or not to delete the raw data included in the project
</Param>

<RequestExample>

```bash Terminal
curl 'https://api.sievedata.com/v1/delete_project' \
  -X POST \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "project_name": "news_media",
    "delete_data": true
  }'
```

</RequestExample>

<ResponseExample>

```json Example
{
  "description": "Deleted project news_media."
}
```

</ResponseExample>

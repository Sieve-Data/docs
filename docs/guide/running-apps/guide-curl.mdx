we will provide a few examples to run the sieve retalker app. the retalker app takes in a video and an audio, then converts (todo: reword).

## Using the curl command

Export your API Key
```sh
export SIEVE_API_KEY=<your api key>
```

Pushing the job

```sh 
$ curl 'https://mango.sievedata.com/v2/push' \
  -H 'X-API-Key: <your api key>' \
  --data-raw '{
    "function": "sieve-internal/video_retalker",
    "inputs": {
      "source_video": {
        "url": "https://storage.googleapis.com/sieve-au-fn-iter-out-us-central1-public-file-upload-bucket/0d5fbb11-5bea-48f8-baf8-28e82c06d31f/d9d382d6-171d-4b79-9ce6-7bfea0691049-input-source_video"
      },
      "target_audio": {
        "url": "https://storage.googleapis.com/sieve-au-fn-iter-out-us-central1-public-file-upload-bucket/0d5fbb11-5bea-48f8-baf8-28e82c06d31f/d9d382d6-171d-4b79-9ce6-7bfea0691049-input-target_audio"
      }
    }
  }' \
```

---
title: "Workflows"
---

A workflow is a list of steps each of which contain [models](/guide/concepts/models). They define how your videos are processed. Sieve is currently invite-only if you'd like to build your own models or workflows. If you're interested in this, please [contact us](https://docs.google.com/forms/d/e/1FAIpQLSdoXPC_FXbp_J-S7mp1-0MPIjXN1x84eY1Rqa6cUjvn-D_HRQ/viewform).

### Create the Sieve function

Let's create a Sieve function with the name `echo_with_exclamation`. That takes in a string, and return the string with an exclamation mark appened at the end.

```python
import sieve

@sieve.function(
    name="echo_with_exclamation",
)
def echo_with_exclamation(s: str) -> str:
    return s + "!"
```

### Running the function

Create another python file to run the function 

```bash
touch main.py
```

Copy paste the following code that runs the sieve function. You can find your org name by running `sieve whoami`.


```python
import sieve

echo_with_exclamation = sieve.function.get(f"<your-org-here>/echo_with_exclamation")
result = echo_with_exclamation.run("hello world")
print(result)
```

Run the file and see the result of the function!

```bash
$ python main.py
hello world!
```

---
title: "Visibility"
description: "Setting the visibiltiy of your function and jobs"
---

Any function deployed to Sieve is private by default and only accessible by you. However, Sieve also let’s you make functions public which let’s multiple users share models and access their web pages with a unique URL. You can make these changes by going to your function settings page.

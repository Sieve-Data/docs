---
title: "Delete Video Data"
api: "POST https://api.sievedata.com/v1/delete_video_data"
---

### Parameters

<Param body="project_name" type="string" required>
  The name of the project
</Param>

<Param body="video_name" type="string" required>
  The name of the video
</Param>

<RequestExample>

```bash Terminal
curl 'https://api.sievedata.com/v1/delete_video_data?project_name=CamNet&video_name=MyVideo' \
  -X DELETE \
  -H "X-API-Key: YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Example
{
  "description": "Deleted video data of MyVideo in project CamNet successfully."
}
```

</ResponseExample>

---
title: "Get Status of Specific Job"
api: "GET https://api.sievedata.com/v1/get_job_status"
---

Get the status of a specific job. The status is either `queued`, `processing`, `finished`, or `failed`.

### Parameters

<Param query="project_name" type="string" required>
  The name of the project
</Param>

<Param query="job_id" type="string" required>
  The job id of a videeo you've submitted
</Param>

<RequestExample>

```bash Terminal
curl 'https://api.sievedata.com/v1/get_job_status?project_name=amazing_project_1&job_id=add3978a-94ac-4585-ab52-8bc5c2295c6f' \
  -X GET \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

</RequestExample>

<ResponseExample>

```json Example
{
  "description": "Successful",
  "status": "finished"
}
```

</ResponseExample>

---
title: "Query Intervals"
api: "POST /query/intervals"
---

<ParamField body="project_name" type="string" required>
  name of Sieve project
</ParamField>

<ParamField body="source_name" type="string" required>
  name of source
</ParamField>

This endpoint lets you retrieve a list of intervals within a source that match a given query.

<CodeGroup>
```json People
{
  "class": "person",
  "project_name": "my_project",
  "source_name": "some_source"
}
```
```json Jumping People
{
  "class": "person",
  "temporal.velocity.y": {
    "$gt": 10
  },
  "project_name": "my_project",
  "source_name": "some_source"
}
```
</CodeGroup>
#### Operations

- [$exists](https://www.mongodb.com/docs/manual/reference/operator/query/exists/)
- [$gte](https://www.mongodb.com/docs/manual/reference/operator/query/gte/)
- [$gt](https://www.mongodb.com/docs/manual/reference/operator/query/gt/)
- [$lte](https://www.mongodb.com/docs/manual/reference/operator/query/lte/)
- [$lt](https://www.mongodb.com/docs/manual/reference/operator/query/lt/)
- [$ne](https://www.mongodb.com/docs/manual/reference/operator/query/ne/)
- [$in](https://www.mongodb.com/docs/manual/reference/operator/query/in/)

<ResponseExample>
```json
{
  "description": "Successful, returned intervals",
  "intervals": [
    [
      0,
      69
    ]
  ],
  "project_name": "some_project",
  "query_latency": "0.098",
  "source_name": "some_source"
}
```
</ResponseExample>

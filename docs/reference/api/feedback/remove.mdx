---
title: "Removals"
api: "POST /feedback/remove_object"
---

<ParamField body="data" type="list" required>
  a list of object_ids to remove
</ParamField>

Each removal is a string specifying the `object_id` of the object to remove.

<RequestExample>
```bash Request
curl "https://v1-api.sievedata.com/v1/feedback/remove_object" \
  -X POST \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
        "data": [
          "some_existing_id",
          "some_other_existing_id",
          ...
        ]
      }'
```
</RequestExample>
<ResponseExample>
```json
{
  "successful_removals": ["some_id"],
  "skipped_removals": []
}
```
</ResponseExample>

---
title: "Additions"
api: "POST /feedback/add_object"
---

<ParamField body="additions" type="list" required>
  a list of additions
</ParamField>

Each addition is a dictionary with the information that you want to instantiate a new object with.

<CodeGroup>
```json New Car
{
  "object_id": "some_id",
  "class": "car",
  "temporal": [
    "frame_number": 0,
    "bounding_box": {
      "x1": 0,
      "y1": 0,
      "x2": 100,
      "y2": 100
    }
  ]
}
```
</CodeGroup>

<RequestExample>
```bash Request
curl "https://v1-api.sievedata.com/v1/feedback/add_object" \
  -X POST \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
        "additions": [
          {
            "object_id": "some_id",
            "class": "car",
            "temporal": [
              "frame_number": 0,
              "bounding_box": {
                "x1": 0,
                "y1": 0,
                "x2": 100,
                "y2": 100
              }
            ]
          },
          ...
        ]
      }'
```
</RequestExample>


<ResponseExample>
```json
{
  "successful_additions": ["some_id"],
}
```
</ResponseExample>

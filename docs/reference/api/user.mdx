---
title: "User Info"
api: "GET /user"
---

This endpoint allows you to access information about the currently authenticated user, including API quotas and usage.

<RequestExample>
```bash Request
curl "https://v1-api.sievedata.com/v1/user" \
  -X GET \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" 
```
</RequestExample>

<ResponseExample>
```json
{
  "API-Key": "YOUR_KEY",
  "hours_left": 3,
  "name": "bob-bob-com"
}
```
</ResponseExample>

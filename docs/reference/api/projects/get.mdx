---
title: "Get Project"
api: "GET /projects/{project_name}"
---

<ParamField path="project_name" type="string" required>
  name of Sieve project
</ParamField>

This endpoint allows you to get a Sieve project and all of its associated data.

<RequestExample>
```bash Request
curl "https://v1-api.sievedata.com/v1/projects/your_project_name" \
  -X GET \
  -H "X-API-Key: YOUR_API_KEY"
```
</RequestExample>
<ResponseExample>
```json
{
  "fps": 30,
  "layers": [
    {
      "iteration": "video",
      "models": [
        {
          "model_id": "some_model_id",
          "model_name": "some_model_name"
        }
      ]
    }
  ],
  "project_name": "your_project_name",
  "store_data": true
}
```
</ResponseExample>

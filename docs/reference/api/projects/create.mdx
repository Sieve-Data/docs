---
title: "Create Project"
api: "POST /projects"
---

<ParamField body="project_name" type="string" required>
  name of Sieve project
</ParamField>

<ParamField body="config" type="dictionary" required>
  <Expandable title="properties">
    <ParamField body="layers" type="list" required>
      A `list` of `SieveLayer` objects.
    </ParamField>

    <ResponseField name="fps" type="number" required>
      The frames per second to process the video at
    </ResponseField>

    <ResponseField name="store_data" type="boolean" required>
      Whether or not to store raw data within Sieve
    </ResponseField>
  </Expandable>
</ParamField>

This endpoint allows you to create a new project within Sieve.

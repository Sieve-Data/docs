---
title: "List Projects"
api: "GET /projects"
---


This endpoint allows you to get all the projects in your account.

<RequestExample>
```bash Request
curl "https://v1-api.sievedata.com/v1/projects" \
  -X GET \
  -H "X-API-Key: YOUR_API_KEY"
```
</RequestExample>

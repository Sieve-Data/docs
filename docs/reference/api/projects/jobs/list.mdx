---
title: "List Jobs"
api: "GET /projects/{project_name}/jobs"
---

<ParamField path="project_name" type="string" required>
  name of Sieve project
</ParamField>

This endpoint allows you to get information all the jobs in a project.

<RequestExample>
```bash Request
curl "https://v1-api.sievedata.com/v1/projects/bob/jobs" \
  -X GET \
  -H "X-API-Key: YOUR_API_KEY"
```
</RequestExample>
